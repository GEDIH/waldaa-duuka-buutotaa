Seenaa fi Kaayyoo / Description & Purpose

Maqaa Abbaa, Kan Ilmaa, Kan Hafuura Qulqulluu, Waaqa Tokko. Ameen.

Sirni Galmee Miseensummaa Waldaa Duukaa Bu'ootaa (WDB) pilaatformii dijitaalaa dha; kaayyoon isaa namoonni dhuunfaa akka miseensa Waldaa Duukaa Bu'ootaa ta'anii galmaa'aniif karaa sirnaawaa fi amantii irratti hundaa'e qopheessuudha.

Sirnichi odeeffannoo miseensotaa haala sirrii fi amanamaan walitti qabuu, iyyannoo miseensummaa sirnaan ilaalu fi galmee miseensotaa eeguudhaan bulchuuf kan tajaajilu dha. Kun hundi sirnaa fi danbii Mana Kiristaanaa Ortodoksii Tawaahidoo kabajuudhaan kan raawwatamu dha.

Pilaatformiin kun adeemsa galmee miseensummaa salphaa fi saffisaa taasisa; akkasumas dhugaa fi sirrii ta'uu odeeffannoo miseensotaa eeguudhaan tokkummaa fi qindoomina miseensota Waldii Duukaa Bu'ootaa cimsuuf gargaara.

Dabalataanis, sirnichi bulchitootaaf carraa to'annoo fi tajaajila gaarii kenna; kunis galmee miseensotaa sirnaan bulchuu, hordofuu fi haaromsuu akka danda'an taasisa.

Sirni Galmee Miseensummaa WDB kun tokkummaa amantootaa cimsuuf, hojii Wangeelaa deeggaruuf, akkasumas dhaloota amantii keessatti guddisuuf teeknooloojii ammayyaa fayyadamuun kan hundaa'e dha.
